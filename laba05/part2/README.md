Отключение системного вызова kexec_load: с помощью модуля sysctl настраивается параметр kernel.kexec_load_disabled на значение "1", что отключает использование системного вызова kexec_load.<b>

Ограничение использования пространств имен пользователей: с помощью модуля sysctl настраивается параметр user.max_user_namespaces на значение "0", что ограничивает использование пространств имен пользователей.<b>

Запрет системного вызова bpf для непривилегированных пользователей: с помощью модуля sysctl настраивается параметр kernel.unprivileged_bpf_disabled на значение "1", что запрещает использование системного вызова bpf для непривилегированных пользователей.<b>
