PART 2:<b>
с помощью ansible, установить nginx и произвести настройку сервера, путем создания конфигурационного файла из шаблона.<b>
Конфигурация может быть дефолтная, но добавить два отличительных параметра в header<b>
X-Frame-Options "SAMEORIGIN"<b>
X-Custom-Header "TestAnsiblePlaybook"<b>

После отработки playbook выполнить команду<b>
curl --head ip<b>
Вывод должен, примерно такой<b>
HTTP/1.1 200 OK<b>
.....<b>
X-Frame-Options: SAMEORIGIN<b>
X-Custom-Header: TestAnsiblePlaybook<b>
....<b>

То есть, что сервер работает и наши настройки применились.<b>
Подсказка, после создания конфигурационного файла из шаблона, перезапустите (тоже из этого же плейбука) nginx, чтобы применилась новая конфигурация.<b>
Совет, чтобы проконтролировать процесс применения конфигурации с помощью ansible, можно выполнить в этом плейбук выполнить команду nginx -t. В результате, Вы без curla, будете сразу знать, применилась Ваша конфигурация или нет.<b>
